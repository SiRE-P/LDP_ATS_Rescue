Repo for DFO-SiRE (Fisheries and Oceans Canada, Salmon in Regional Ecosystems Program) lake acoustic and trawl survey data from 1977-2007 in British Columbia lakes, targeting juvenile Sockeye salmon. These data were cleaned and extracted by data interns working in the Canadian Institute of Ecology and Evolution (CIEE)'s Living Data Project (LDP) program, overseen by postdoctoral fellows from the same program, and DFO staff.

The ACOUSTIC_TARGETS folder includes input data, R code and the cleaned outputs of lake acoustic sounding data. It was led by Yuliya Shtymburska (intern, University of Regina) and Sandra Emry (UBC postdoctoral fellow). This work is complete; its data and documentation will likely be the first of these three folders that will be posted on the Government of Canada's Open Data portal.

The TRAWL_BIOSAMPLE folder is work is in progress. It includes input data, R code and the cleaned outputs of lake trawl biosample data. It was led by Alice Assmar (intern, McGill University) and David Hunt (postdoctoral fellow, McGill University). After this internship was complete, Alice continued to work on the materials in this folder as part of the DFO data report work (see next folder description). This work is ongoing and this folder is planned to be updated later in 2026.

The DFO_ATS_REPORT folder is the workspace for a report in the Canadian Data Report of Fisheries and Aquatic Sciences series that will include the acoustic target and trawl biosample data, which are related, and are best understood in conjunction with each other. It is work that is led by Alice Assmar with the assistance of David Hunt. This work is ongoing and this folder is planned to be updated later in 2026.

All of these data will be available in the near future at the Government of Canada's Open Data portal, and the Data Report will likely be published by DFO Library in late 2026 or early 2027.
